# Bienvenido a  From 0 To Cloud. 
## Generación de un catálogo de componentes en Storybook

Storybook es una herramienta que nos permite la creación, exposición y configuración de componentes gráficos aislados del entorno donde se desarrollan. La idea principal es facilitar el aprovechamiento, desarrollo y testing de cada uno de los elementos que conforman la aplicación.

Este proyecto creado para continuar con la línea editorial **From 0 To Cloud** pretende dar una funcionalidad más a realizar en la construcción de aplicaciones capaces de ser portadas a la nube, siendo estas independientes y funcionales en cualquier entorno.
