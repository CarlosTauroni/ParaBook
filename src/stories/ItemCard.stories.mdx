# COMPONENTE ELEMENTO TARJETA

Modulo:

- El import para poder cargar el componente es el siguiente:

```js live=true
import ItemCard from '../components/ItemCard/ItemCard.vue';
```

Componente:

- Para incorporar el componente a HTML, el prefijo a utilizar para la etiqueta HTML es:

```html live=true
<ItemCard :data=this.data :hasError=this.hasError />
``` 
El elemento tarjeta no tiene más variantes, implementa el elemento Button internamente que es descrito en esta sección.

- data: ([{}]) Vector de Jsons de producto, por cada uno de los elementos añadidos se almacenna la información del producto.
- hasError: (Boolean) Flag que cambia la visualización del elemento en caso de que el servicio no sea capaz de devolver información.

El cuerpo de cada elemento del Json corresponde con los campos enviados desde el backend y deben contener los siguientes campos:

```js
{
  amount: Number,
  alttext: String,
  description: String,
  id: String,
  img: Image,
  name: String,
  price: Number
}
```
