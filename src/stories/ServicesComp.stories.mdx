# COMPONENTE DE OFERTA DE SERVICIOS

Constructo compuesto de:

- Título de sección
- Botón

Modulo:

- El import para poder cargar el componente es el siguiente:

```js live=true
import Services from '../pages/ServicesComp/ServicesComp.vue';
```

Componente:

- Para incorporar el componente a HTML, el prefijo a utilizar para la etiqueta HTML es:

```html live=true
<Services :text=this.text />
``` 
El componente de ofertas de servicios no tiene más variantes.

- text: (String) Texto que aparecerá en el cuerpo del componente anunciando un servicio.
