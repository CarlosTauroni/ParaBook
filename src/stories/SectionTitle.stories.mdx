# COMPONENTE TÍTULO DE SECCIÓN

Modulo:

- El import para poder cargar el componente es el siguiente:

```js live=true
import SectionTitle from '../components/SectionTitle/SectionTitle.vue';
```

Componente:

- Para incorporar el componente a HTML, el prefijo a utilizar para la etiqueta HTML es:

```html live=true
<SectionTitle :msg=this.msg />
``` 
El título de sección no tiene más variantes.

- msg: (String) Título que tendrá la sección.
