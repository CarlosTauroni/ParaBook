# COMPONENTE CARRITO DE LA COMPRA

Constructo compuesto de:

- ShoppingCart

Modulo:

- El import para poder cargar el componente es el siguiente:

```js live=true
import CartIcon from '../components/CartIcon/CartIcon.vue';
```

Componente:

- Para incorporar el componente a HTML, el prefijo a utilizar para la etiqueta HTML es:

```html live=true
<CartIcon :showCart=this.showCart :totalCart=this.totalCart :cart='this.cart' />
```

El elemento botón de carrito de la compra no tiene más variantes e implementa el elemento ShoppingCart descrito en esta misma sección.

Los argumentos que recibe son los siguientes:

- showCart: (Boolean) Indica si el carro estará abierto o cerrado.
- totalCart: (Number) Indica la cantidad contenida por el carrito, esta cantidad ha de ser enviada calculada.
- cart : [{}] Vector de Jsons de producto, por cada uno de los elementos añadidos se almacenna la información del producto.

El cuerpo de cada elemento del Json corresponde con los campos enviados desde el backend y deben contener los siguientes campos:

```js
{
  amount: Number,
  alttext: String,
  description: String,
  id: String,
  img: Image,
  name: String,
  price: Number
}
```
