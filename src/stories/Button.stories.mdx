# COMPONENTE BOTÓN

Modulo:

- El import para poder cargar el componente es el siguiente:

```js live=true
import Button from "../components/Button/Button.vue';
```

Componente:

- Para incorporar el componente a HTML, el prefijo a utilizar para la etiqueta HTML es:

```html live=true
<Button :msg="this.msg" :size="this.size" :type="this.type" @click="onClick()" />
```

El elemento botón con tiene tres variantes. Estas son:

- Button Primary
- Button Secondary
- Button Tertiary

Todos los tipos de componentes botón aceptarán los siguientes parámetros:

- msg: (String) Texto que figurará en el botón.
- size: ['small' | 'large'] Tamaño del botón a incluir.
- type: ['primary' | 'secondary' | 'tertiary'] Tipología de botón a incluir.
- @click: El parámetro pasado ha de ser una función definirda en el componente llamante.
