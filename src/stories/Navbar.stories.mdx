# COMPONENTE BARRA DE NAVEGACIÓN

Modulo:

- El import para poder cargar el componente es el siguiente:

```js live=true
import Navbar from '../components/Navbar/Navbar.vue';
```

Componente:

- Para incorporar el componente a HTML, el prefijo a utilizar para la etiqueta HTML es:

```html live=true
<Navbar :items=this.items />
``` 
La barra de navegación no tiene más variantes.

- items: (Array) Elementos en un Array de JSON, que se mostraránen la barra de navegación.

El cuerpo de cada elemento del Json corresponde con las pestañas que va a presentar la aplciación, debiendo contener los siguientes campos:

```js
{
  id: String,
  name: String,
  tab: String,
}
```
